Projects

Below are some of the key projects I've worked on, showcasing my practical skills in embedded systems, software development, and hardware integration.

1. Smart Home Automation Hub

Technologies: ESP32, MQTT, Python, Node-RED, Raspberry Pi

Description: Developed a custom smart home hub capable of controlling various IoT devices (lights, sensors, smart plugs) using a centralized system. Features included a web-based interface for remote control, integration with voice assistants, and data logging for environmental monitoring.

  • Implemented secure MQTT communication between devices and the central hub.
  • Designed a modular architecture allowing easy integration of new devices.
  • Utilized Node-RED for visual programming of automation flows.

2. Autonomous Line-Following Robot

Technologies: STM32 Microcontroller, PID Control, IR Sensors, Motor Drivers, C/C++

Description: Designed and built an autonomous robot that follows a black line on a white surface using PID (Proportional-Integral-Derivative) control for precise steering. The project involved low-level microcontroller programming, sensor interfacing, and real-time control algorithm implementation.

  • Developed efficient interrupt-driven sensor reading routines.
  • Fine-tuned PID parameters for optimal performance on various curves.
  • Designed a custom PCB for power management and sensor integration.

3. Custom Wearable Health Monitor

Technologies: Arduino Nano, PPG Sensor, Bluetooth LE, Android App (Java)

Description: Prototyped a wearable device that measures heart rate and blood oxygen saturation (SpO2) using a photoplethysmography (PPG) sensor. Data is transmitted via Bluetooth Low Energy (BLE) to a custom Android application for real-time visualization and data logging.

  • Implemented signal processing algorithms for robust heart rate detection.
  • Developed a low-power BLE communication protocol.
  • Designed a user-friendly Android interface for data display.
