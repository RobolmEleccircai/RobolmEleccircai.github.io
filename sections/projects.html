3. Intelligent Traffic - STM32-based Dual-Mode Vehicle Positioning and Obstacle Detection System View on GitHub

Project Time: Aug. 2022

Project Description: Designed a GPS + BDS dual-mode positioning and obstacle detection system based on STM32, allowing users to obtain vehicle positioning information through the upper computer. System Design Includes:

  • Sensor Selection: Includes STM32 processor, BDS + GPS positioning module, laser distance measurement sensor, etc.
  • Circuit Design: Includes MCU circuit, indicator circuit, LED circuit, interface circuit, etc.
  • Software Design: Includes positioning mode selection and display of positioning information.

5. Balancing Car Intelligent Posture Control System Design View on GitHub

Project Time: Dec. 25, 2022

Project Description: Designed a balancing car posture measurement and real-time control system based on the ESP32 microcontroller. The system uses the MPU6050 gyroscope to detect the tilt angle and angular velocity of the car in real time, and adjusts the motor output using a PID control algorithm to achieve self-balancing.

Project Features:

  • Real-time sensing of the tilt angle and angular velocity of the car;
  • Stabilization control through PID algorithm (PD for upright control and PI for speed control);
  • ESP32 control core processes sensor data and drives dual motors;
  • Supports Bluetooth wireless communication for remote control of the car’s motion direction;
  • Provides a graphical Bluetooth control interface (based on MFC development).

Key Technologies:

  • MPU6050 gyroscope module for angle and angular velocity detection;
  • PID control to dynamically adjust motor PWM output based on posture data;
  • Motor driver module: DRV8833, Power supply module: LM2596S for voltage regulation;
  • Bluetooth serial communication to implement human-machine interaction, device scanning, command sending, and motion control;
  • Reasonable hardware structure to ensure stable operation and performance.
